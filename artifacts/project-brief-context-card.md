# Project Brief: Context Card

> Pre-Research Stage — Everything here reflects initial assumptions and early thinking. Nothing has been validated yet.

## Context
The Context Card is a core UI component within the Context Binding system in the Wix editor. It lives in the Section Settings Panel and represents an attached data context — showing its name, type, source, usage status, and exposed fields/actions. This work is about exploring alternative visual representations of the Context Card to validate design ideas before committing to a direction.

## Direction
Create multiple UX concept options for how the Context Card could be represented in the Settings Panel. The goal is to explore different visual and interaction approaches, then evaluate which best serves the use case.

## Goal
Validate an idea — explore UX variations for the Context Card representation and assess which direction works best.

## Target Group
Partners (agencies/freelancers building sites for clients)
